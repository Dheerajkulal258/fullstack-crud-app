# 🛒 Full Stack CRUD App

A complete full-stack application using:

## 🚀 Tech Stack
- Node.js
- Express.js
- MongoDB
- HTML, CSS, JavaScript

## ✨ Features
- Add Items
- View Items
- Update Items
- Delete Items
- Loading UI & Animations

## 📂 Project Structure
- Backend (API)
- Frontend (public folder)

## ▶️ Run Locally
npm install
npm run dev

🌐 Open
http://localhost:5000
